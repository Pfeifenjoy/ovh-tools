## ✨ Features

- 🚀 Simplified OVH API interactions
- 🔧 Developer-friendly command interface
- 📦 TypeScript support with full type safety
- 🎯 Optimized for modern development workflows

## 🚀 Installation

```bash
npm install -g ovh-tools
```

## 📖 Usage

```bash
ovh-tools --help
```

## 🛠️ Development

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test

# Format code
npm run format
```
